# v-model



## 1. v-model底层原理

```vue
<template>
  <div>
    <h1>{{ message }}</h1>
    <input type="text" v-model="message" />
    <input type="text" v-bind:value="message" v-on:input="handleInput" />
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: "hello world",
    };
  },
  methods: {
    handleInput(event) {
      const inputElement = event.target;
      const value = inputElement.value;
      this.message = value;
    },
  },
};
</script>
```
