### v-on监听事件

```vue
<!--监听事件的三种方法-->
<template>
  <div>
    <button v-on:click="add(3)">Add 1</button>
      <!-- v-on:click="add1" -->
      <!-- @click="counter+=3"-->
  </div>
</template>

<script>
export default {
  data() {
    return {
      counter: 0,
    };
  },
  methods: {
      
      add1(){
          this.counter+=2
      }
    // this => 自动指向组件实例
    add(num) {
      this.counter += num;
    },
  },
};
</script>

```

### 事件修饰符

```vue
<!-- 阻止单击事件冒泡 -->
<a @click.stop="doThis"></a>

<!-- 阻止浏览器默认行为 -->
<form @submit.prevent="onSubmit"></form>

```
